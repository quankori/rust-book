# Learning Rust

## Rust

### Install

```bash
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

### Check version

```bash
rustc --version
```

### Run program

```bash
rustc main.rs
```

## Cargo

### Check version

```bash
cargo --version
```

### Create project

```bash
cargo new cargo-projects
```

### Run program

```bash
cargo run
```

```bash
cargon build
```

```bash
cargo check
```
